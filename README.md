# Flowline

Flowline is an open-source industrial runtime for defining, executing, and monitoring
process flows on the shopfloor.

## Scope
- Execution of deterministic process flows
- Modular, testable core runtime
- Clear separation of runtime, UI, and integration layers
- Designed for industrial automation and Poka-Yoke use cases

## Non-Goals
- No reuse of legacy or proprietary systems
- No vendor-specific assumptions
- No hidden business logic

## Project Origin
Flowline is a clean-room implementation created from scratch.
It is not derived from or affiliated with any proprietary system or former employer software.
